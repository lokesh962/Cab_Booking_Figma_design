# 🚗 Car Rental UI – Figma Design

A clean and modern **Car Rental Web UI** designed in Figma. This project showcases a multi-page user interface for browsing, searching, and booking rental cars with an emphasis on simplicity and usability.

---

## 🔗 Figma Design
Access the full design here:  
https://www.figma.com/design/Wea3yZezRh7epzYK7aG5IG/Untitled?node-id=0-1&p=f&t=n6U4Tg6ZmKfVRwcg-0

---

## 📌 Features

- 🏠 Homepage  
  - Hero section with call-to-action  
  - Quick car search (location, date, return date)  
  - Clean navigation bar  

- 🚘 Featured Cars Section  
  - Car cards with model, price, fuel type, seating  
  - “Book Now” button  

- 💰 Budget Filter  
  - Price slider  
  - Manual input option  

- 🔐 Authentication Pages  
  - Login form  
  - Signup form
